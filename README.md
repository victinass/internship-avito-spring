### Запуск:

##### 1. Склонировать репозиторий с помощью команды
```bash
git clone 'https://github.com/victinass/internship-avito-spring.git'
```
#### 2. Перейти в папку проекта
```bash
cd Task_2
```
#### 3. Установить зависимости если нужно
```bash
npm install
```
Удостовериться, что установились браузеры
```bash
npx playwright install
```
#### 4. Тесты
- Запустить все тесты:

```bash
npx playwright test
```

- Запустить конкретный тест по названию:
```bash
npx playwright test -g "Валидация фильтра со значением границ наоборот"
```
